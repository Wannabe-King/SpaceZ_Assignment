# spacez_assignment

This is the implementation of Spacez Coupon Page

## Getting Started

Clone The Repo:

```
git clone https://github.com/Wannabe-King/SpaceZ_Assignment
```

Get all the dependencies:

```
flutter pub get
```

Run the project

```
flutter run
```
